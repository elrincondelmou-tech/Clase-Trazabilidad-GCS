# Clase-Trazabilidad-GCS
El sistema debe permitir al usuario capturar y adjuntar una fotografía del bache o accidente vial directamente desde la cámara del dispositivo móvil al momento de realizar un reporte.

Este proyecto documenta la trazabilidad y configuración del software.
